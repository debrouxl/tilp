2. Overview

2.1 Standard GUI

When you launch TiLP2, you will see something like this:



All operations/settings can be done by the top menu bar (not shown here), Drag & Drop, or by popup menus (right click on each view).

By default, the window is split into 2 parts:
- the local view (right frame) : contains a toolbar (calculator-related operations), a tree and an information frame (current folder);
- the remote view (left frame) : contains a toolbar (computer-related operations), a list and an information frame (memory used and/or free).

Any operation is based on the content of the left or the right frame. This is the same behaviour as TiLP I.
